LINUX UBUNTU

You can install Git on Linux through the package management tool that comes with your distribution.
Debian/Ubuntu

1    Git packages are available using apt.

2    It's a good idea to make sure you're running the latest version. To do so, Navigate to your command 
	prompt shell and run the following command to make sure everything is up-to-date: sudo apt-get update.

3    To install Git, run the following command: sudo apt-get install git-all.

4    Once the command output has completed, you can verify the installation by typing: git version
